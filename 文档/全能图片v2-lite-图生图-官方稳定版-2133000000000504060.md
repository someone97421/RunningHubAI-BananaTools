## 简介

通过文本指令变换上传图片，支持快速 Prompt 引导编辑、视觉精修与创意改动，低延迟。开箱即用的 REST 推理 API，性能优异、无冷启动、价格实惠。

## 模型特点

- 文本指令编辑
- Prompt 引导
- 视觉精修
- 创意改动
- 低延迟

## 提交请求

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/rhart-image-n-g31-flash-lite-official/image-to-image' \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "imageUrls": [
    "https://example.com/input.png"
  ],
  "prompt": "将背景替换为阳光海滩，保持主体不变。",
  "aspectRatio": "auto"
}'
```

## 请求参数

| 参数 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| `imageUrls` | List(String) | 必填 | Input image URLs for editing.；最多 4 项；单项最大 10 MB；格式：["JPG","PNG"]；默认值：示例文件 URL |
| `prompt` | String | 必填 | Edit instruction describing how to transform the input image.；文本长度：1–20000；默认值：将背景替换为阳光海滩，保持主体不变。 |
| `aspectRatio` | String | 可选 | Output aspect ratio (maps to API size). Default auto.；枚举：auto, 1:1, 16:9, 9:16, 4:3, 3:4, 3:2, 2:3, 5:4, 4:5, 21:9, 4:1, 1:4, 8:1, 1:8；默认值：auto |
| `webhookUrl` | String | 可选 | 任务完成时接收 POST 回调的地址 |

## 文件上传

媒体参数可使用公共 URL、Base64 data URI，或先上传本地文件：

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/media/upload/binary' \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --form "file=@/path/to/input.png"
```

上传链接有效期通常为 1 天。
