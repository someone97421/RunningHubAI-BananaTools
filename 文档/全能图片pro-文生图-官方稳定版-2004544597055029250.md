## 提交请求

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/rhart-image-n-pro-official/text-to-image' \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "resolution": "1k",
  "prompt": "在一片广阔无垠的大海边，一只快乐的猴子坐在沙滩上，享受着温暖的阳光。它的旁边有一个小小的香蕉，象征着自然的馈赠。天空湛蓝，阳光明媚，海浪轻拍沙滩，营造出轻松愉快的氛围。整体风格为手绘插画，色彩明亮鲜艳，突出积极向上的情绪，让观看者感受到幸福和安宁。",
  "aspectRatio": "3:4"
}'
```

## 请求参数

| 参数 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| `resolution` | String | 必填 | resolution；枚举：1k, 2k, 4k；默认值：1k |
| `prompt` | String | 必填 | prompt；文本长度：1–20000；默认值：在一片广阔无垠的大海边，一只快乐的猴子坐在沙滩上，享受着温暖的阳光。它的旁边有一个小小的香蕉，象征着自然的馈赠。天空湛蓝，阳光明媚，海浪轻拍沙滩，营造出轻松愉快的氛围。整体风格为手绘插画，色彩明亮鲜艳，突出积极向上的情绪，让观看者感受到幸福和安宁。 |
| `aspectRatio` | String | 可选 | aspectRatio；枚举：1:1, 3:2, 2:3, 3:4, 4:3, 4:5, 5:4, 9:16, 16:9, 21:9；默认值：3:4 |
| `webhookUrl` | String | 可选 | 任务完成时接收 POST 回调的地址 |

```bash

  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
```

常见状态：`QUEUED`、`RUNNING`、`SUCCESS`、`FAILED`。结果位于 `results`，其中 `url` 通常仅在 24 小时内有效，应及时下载或转存。

## 文件上传

媒体参数可使用公共 URL、Base64 data URI，或先上传本地文件：

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/media/upload/binary' \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --form "file=@/path/to/input.png"
```

上传链接有效期通常为 1 天。
