## 提交请求

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/rhart-image-g-2-official/image-to-image' \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "prompt": "将这个客厅彻底改造为“植物园温室”风格。把原有的沙发替换成复古的绿色天鹅绒材质，墙面变成做旧的红砖墙。整个房间内挂满茂盛的热带藤蔓植物，阳光透过巨大的玻璃穹顶洒在地毯上。保持房间原本的物理空间大小、门窗位置以及家具的摆放结构完全不变。",
  "imageUrls": [
    "https://example.com/input.png"
  ],
  "aspectRatio": "16:9",
  "resolution": "2k",
  "quality": "medium"
}'
```

## 请求参数

| 参数 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| `prompt` | String | 必填 | 编辑指令描述；文本长度：1–20000；默认值：将这个客厅彻底改造为“植物园温室”风格。把原有的沙发替换成复古的绿色天鹅绒材质，墙面变成做旧的红砖墙。整个房间内挂满茂盛的热带藤蔓植物，阳光透过巨大的玻璃穹顶洒在地毯上。保持房间原本的物理空间大小、门窗位置以及家具的摆放结构完全不变。 |
| `imageUrls` | List(String) | 必填 | 参考图片（1-10张）；最多 10 项；单项最大 10 MB；格式：["JPG","JPEG","PNG","WEBP"]；默认值：示例文件 URL |
| `aspectRatio` | String | 可选 | 输出图像宽高比；枚举：1:1, 1:2, 2:1, 1:3, 3:1, 2:3, 3:2, 3:4, 4:3, 4:5, 5:4, 9:16, 21:9, 9:21, 16:9；默认值：16:9 |
| `resolution` | String | 必填 | 输出图像分辨率档位；枚举：1k, 2k, 4k；默认值：2k |
| `quality` | String | 必填 | quality；枚举：low, medium, high；默认值：medium |
| `webhookUrl` | String | 可选 | 任务完成时接收 POST 回调的地址 |

## 文件上传

媒体参数可使用公共 URL、Base64 data URI，或先上传本地文件：

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/media/upload/binary' \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --form "file=@/path/to/input.png"
```

上传链接有效期通常为 1 天。
