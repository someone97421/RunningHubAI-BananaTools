## 提交请求

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/rhart-image-g-2/image-to-image' \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "prompt": "在马克杯的正中央，添加一个精致的几何风格狐狸 Logo，Logo 下方清晰地印着文字 \"Wild Fox\"。请保持原图的光影结构和陶瓷质感完全不变。 然后生成一张产品介绍说明书，比例为21：9",
  "imageUrls": [
    "https://example.com/input.png"
  ],
  "aspectRatio": "16:9",
  "resolution": "1k"
}'
```

## 请求参数

| 参数 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| `prompt` | String | 必填 | prompt；文本长度：1–20000；默认值：在马克杯的正中央，添加一个精致的几何风格狐狸 Logo，Logo 下方清晰地印着文字 "Wild Fox"。请保持原图的光影结构和陶瓷质感完全不变。 然后生成一张产品介绍说明书，比例为21：9 |
| `imageUrls` | List(String) | 必填 | imageUrls；最多 10 项；单项最大 30 MB；格式：["JPG","PNG"]；默认值：示例文件 URL |
| `aspectRatio` | String | 可选 | aspectRatio；枚举：1:1, 2:3, 3:2, 4:5, 5:4, 4:3, 3:4, 16:9, 9:16, 21:9, 9:21, 2:1, 1:2, 3:1, 1:3；默认值：16:9 |
| `resolution` | String | 可选 | 可选择 1k/2k/4k，因接口稳定性限制，暂不保证精准输出 2k/4k 分辨率，多数情况下仍会输出 1k 图像，介意请使用官方接口。；枚举：1k, 2k, 4k；默认值：1k |
| `webhookUrl` | String | 可选 | 任务完成时接收 POST 回调的地址 |

## 文件上传

媒体参数可使用公共 URL、Base64 data URI，或先上传本地文件：

```bash

  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --form "file=@/path/to/input.png"
```

上传链接有效期通常为 1 天。
