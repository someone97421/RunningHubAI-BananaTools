## 提交请求

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/rhart-image-n-pro/edit' \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "imageUrls": [
    "https://example.com/input.png"
  ],
  "prompt": "基于原图风格，将老奶奶替换为一只年迈慈祥的猴子奶奶，她穿着格子围裙，正用香蕉制作晚餐，锅里是香蕉炖菜或香蕉泥。厨房环境保持不变：木制家具、悬挂的辣椒大蒜、陶罐、篮子里的新鲜蔬菜。窗外阳光温暖，整体氛围依旧温馨宁静。风格为手绘水彩插画，色彩柔和，细节丰富，充满家庭温情。",
  "aspectRatio": "3:4",
  "resolution": "1k"
}'
```

## 请求参数

| 参数 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| `imageUrls` | List(String) | 必填 | imageUrls；最多 10 项；单项最大 10 MB；格式：["JPG","PNG"]；默认值：示例文件 URL |
| `prompt` | String | 必填 | prompt；文本长度：1–20000；默认值：基于原图风格，将老奶奶替换为一只年迈慈祥的猴子奶奶，她穿着格子围裙，正用香蕉制作晚餐，锅里是香蕉炖菜或香蕉泥。厨房环境保持不变：木制家具、悬挂的辣椒大蒜、陶罐、篮子里的新鲜蔬菜。窗外阳光温暖，整体氛围依旧温馨宁静。风格为手绘水彩插画，色彩柔和，细节丰富，充满家庭温情。 |
| `aspectRatio` | String | 可选 | 不传 aspectRatio 参数时为自适应图片尺寸；枚举：1:1, 16:9, 9:16, 4:3, 3:4, 3:2, 2:3, 5:4, 4:5, 21:9；默认值：3:4 |
| `resolution` | String | 必填 | resolution；枚举：1k, 2k, 4k；默认值：1k |
| `webhookUrl` | String | 可选 | 任务完成时接收 POST 回调的地址 |

```bash
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \

```

常见状态：`QUEUED`、`RUNNING`、`SUCCESS`、`FAILED`。结果位于 `results`，其中 `url` 通常仅在 24 小时内有效，应及时下载或转存。

## 文件上传

媒体参数可使用公共 URL、Base64 data URI，或先上传本地文件：

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/media/upload/binary' \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --form "file=@/path/to/input.png"
```

上传链接有效期通常为 1 天。
