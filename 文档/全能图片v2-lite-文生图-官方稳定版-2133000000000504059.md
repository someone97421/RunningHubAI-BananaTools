基于文本 Prompt 生成高质量图像，低延迟、灵活宽高比，适合创意与生产场景的快速出图。开箱即用的 REST 推理 API，性能优异、无冷启动、价格实惠。

- 低延迟
- 灵活宽高比
- 高质量出图

- 无冷启动

## 提交请求

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/rhart-image-n-g31-flash-lite-official/text-to-image' \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "prompt": "黄昏时分的山峦风景，柔和金色光线，画面宁静优美。",
  "aspectRatio": "1:1"
}'
```

## 请求参数

| 参数 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| `prompt` | String | 必填 | Text prompt describing the image to generate.；文本长度：1–20000；默认值：黄昏时分的山峦风景，柔和金色光线，画面宁静优美。 |
| `aspectRatio` | String | 可选 | Output aspect ratio (maps to API size).；枚举：1:1, 16:9, 9:16, 4:3, 3:4, 3:2, 2:3, 5:4, 4:5, 21:9, 4:1, 1:4, 8:1, 1:8；默认值：1:1 |
| `webhookUrl` | String | 可选 | 任务完成时接收 POST 回调的地址 |

## 文件上传


上传链接有效期通常为 1 天。
