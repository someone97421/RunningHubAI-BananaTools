## 简介

多参考图图像编辑接口，单次最多上传 10 张参考图，支持原图结构锁定、像素级局部重绘、风格迁移、分层透明素材导出，输出 2K 高清 PNG/JPG，适配人像改风格、产品改版、多素材融合、老照片修复。

## 模型特点

- 单次最多上传 10 张参考图｜原图结构锁定不畸变｜可调重绘强度｜PNG 透明图层导出

```bash

  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "prompt": "横版复古馆藏风美妆宣传海报，藏青深蓝色哑光底，金色细线条分割拼贴版式，画面中心放置透明滴管 5% 烟酰胺亮肤精华瓶，搭配泛黄复古手稿、香氛线稿、茉莉花卉素描、复古静物老照片做拼贴装饰，烫金字体排版，主标题大字「焕亮透亮・烟酰胺精华」，底部留白标注产品核心功效，复古做旧纸张撕边纹理，轻奢高级美术馆展览质感，柔和暖调光影，8K 商业平面广告，层次丰富精致",
  "width": 1024,
  "height": 1024,
  "imageUrls": [
    "https://example.com/input.png"
  ],
  "resolution": "2k",
  "outputFormat": "jpeg"
}'
```

| 参数 | 类型 | 是否必填 | 说明 |
|---|---|---|---|

## 文件上传

媒体参数可使用公共 URL、Base64 data URI，或先上传本地文件：

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/media/upload/binary' \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --form "file=@/path/to/input.png"
```

上传链接有效期通常为 1 天。
