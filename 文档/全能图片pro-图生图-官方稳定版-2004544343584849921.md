## 提交请求

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/rhart-image-n-pro-official/edit' \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "imageUrls": [
    "https://example.com/input.png"
  ],
  "prompt": "海边沙滩变成夏日祭典现场：猴子戴着纸折小帽，小香蕉插着蜡烛当作“生日蛋糕”，周围有彩旗、西瓜、贝壳风铃。其他小猴子在远处玩耍，烟花在晴空中绽放。风格欢乐卡通，色彩缤纷。",
  "resolution": "1k",
  "aspectRatio": "3:4"
}'
```

## 请求参数

| 参数 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| `imageUrls` | List(String) | 必填 | imageUrls；最多 10 项；单项最大 10 MB；格式：["JPG","PNG"]；默认值：示例文件 URL |
| `prompt` | String | 必填 | prompt；文本长度：1–20000；默认值：海边沙滩变成夏日祭典现场：猴子戴着纸折小帽，小香蕉插着蜡烛当作“生日蛋糕”，周围有彩旗、西瓜、贝壳风铃。其他小猴子在远处玩耍，烟花在晴空中绽放。风格欢乐卡通，色彩缤纷。 |
| `resolution` | String | 必填 | resolution；枚举：1k, 2k, 4k；默认值：1k |
| `aspectRatio` | String | 可选 | 不传 aspectRatio 参数时为自适应图片尺寸；枚举：1:1, 3:2, 2:3, 3:4, 4:3, 4:5, 5:4, 9:16, 16:9, 21:9；默认值：3:4 |
| `webhookUrl` | String | 可选 | 任务完成时接收 POST 回调的地址 |

```bash
  --header "Content-Type: application/json" \

```

## 文件上传

媒体参数可使用公共 URL、Base64 data URI，或先上传本地文件：

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/media/upload/binary' \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --form "file=@/path/to/input.png"
```

上传链接有效期通常为 1 天。
