## 简介

专注于极致的视觉控制与深度图像编辑。它不仅支持基础的单图重绘，更开放了强大的多图特征融合与“参考图生组图”功能。无论是提取多张图片的风格、材质与元素进行无缝重塑，还是基于单张/多张参考图横向扩展出具有完美一致性的视觉系列，该接口都能在保持原始参考特征的前提下，实现极高自由度的艺术加工与工业级设计交付。

## 模型特点

- 深度单图重绘
- 多图特征无缝融合
- 参考图生连贯组图
- 极致主体一致性保障
- 精准材质与光影重塑
- 复杂视觉元素提取

## 提交请求

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/seedream-v5-lite/image-to-image' \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "prompt": "请融合输入的图1与图2，生成一组共3张连贯的国漫风格（Guoman style）横版概念设计图。画面主体为青年方文与他的宠物黑羽乌鸦。第一张：方文穿着图1中锦衣卫风格的服饰，站在图2那座阴雨连绵的古寺大门前，乌鸦在雨中盘旋。第二张：视角拉近，方文推开古寺斑驳的木门，微弱的光影从门缝折射在他脸上。第三张：方文进入古寺内部，乌鸦停在破败的佛像上，周围弥漫着神秘的雾气。要求： 完美提取并融合图1的服装材质细节与图2的环境氛围，3张图需保持完美的主角特征与场景一致性。",
  "width": 2048,
  "height": 2048,
  "imageUrls": [
    "https://example.com/input.png"
  ],
  "sequentialImageGeneration": "disabled",
  "maxImages": 1
}'
```

## 文件上传

```bash

  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --form "file=@/path/to/input.png"
```

上传链接有效期通常为 1 天。
