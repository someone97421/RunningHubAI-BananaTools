# seedream-v5-pro-文生图

- 官方详情页：[https://www.runninghub.ai/call-api/api-detail/2133000000000504073](https://www.runninghub.ai/call-api/api-detail/2133000000000504073)
- 模型 ID：`2133000000000504073`
- 接口：`/seedream-v5-pro/text-to-image`
- 更新时间：2026-07-09 21:20:31

## 简介

纯文本驱动专业商用图像生成接口，无需上传参考图，支持 1024–2048 自定义分辨率，输出 JPG/PNG 透明通道图，文字渲染精准，适配海报、产品、信息长图等商业设计生产。

## 模型特点

- 单次仅生成 1 张图｜输出 PNG/JPG 双格式｜最高 2K 高清分辨率｜原生多语种文字精准渲染

## 提交请求

```bash
curl --location --request POST 'https://www.runninghub.ai/openapi/v2/seedream-v5-pro/text-to-image' \
  --header "Content-Type: application/json" \
  --header "Authorization: Bearer ${RUNNINGHUB_API_KEY}" \
  --data-raw '{
  "prompt": "高端护肤品电商主图，纯白柔光展台，玻璃精华瓶，商业写实摄影，柔和侧逆光，干净无杂物。16:9",
  "width": 1024,
  "height": 1024,
  "resolution": "2k",
  "outputFormat": "jpeg"
}'
```

## 请求参数

| 参数 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| `prompt` | String | 必填 | prompt；文本长度：5–2000；默认值：高端护肤品电商主图，纯白柔光展台，玻璃精华瓶，商业写实摄影，柔和侧逆光，干净无杂物。16:9 |
| `width` | Int | 可选 | width；范围：240–8192；步长：1；默认值：1024 |
| `height` | Int | 可选 | height；范围：240–8192；步长：1；默认值：1024 |
| `resolution` | String | 可选 | 1、优先级高于widthxheight，传递resolution则使用resolution，不再使用widthxheight。；枚举：1k, 2k；默认值：2k |
| `outputFormat` | String | 可选 | 输出图片格式，可选 png / jpeg；枚举：jpeg, png；默认值：jpeg |
| `webhookUrl` | String | 可选 | 任务完成时接收 POST 回调的地址 |
